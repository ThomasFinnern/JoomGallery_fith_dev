### User panel tab details

The controls in details are standard so no further documentation  
![Tab details](https://github.com/ThomasFinnern/JoomGallery_fith_dev/blob/main/.jg_dev_doc/jg_4x/images/site.UserUpload/menu_userpanel_details.en.png?raw=true  "Tab details]")

### User panel tab options

![Tab options](https://github.com/ThomasFinnern/JoomGallery_fith_dev/blob/main/.jg_dev_doc/jg_4x/images/site.UserUpload/menu_userpanel_options.en.png?raw=true "Tab details")

(1) Display user limits  
Display an area where actual count of images/categories are matched with config settings  
![Tab details](https://github.com/ThomasFinnern/JoomGallery_fith_dev/blob/main/.jg_dev_doc/jg_4x/images/site.UserUpload/userpanel_user_limits.en.png?raw=true  "Tab details")  
Not checked as such: The code is not ready to check the limits. so there is no error when limit is exceeded. It may be developed soon.  
A red zone means that the limit has been exceeded.

(2) Display user information   
Display an area with actual count of images/categories. Only visible when user limits is off.  
![user info](https://github.com/ThomasFinnern/JoomGallery_fith_dev/blob/main/.jg_dev_doc/jg_4x/images/site.UserUpload/userpanel_user_info.en.png?raw=true  "user info")  

(3) Display latest user categories   
Display an area with latest user categories. See documentation user panel (below)    

(4) Number of latest categories   
Number of items shown in table of latest categories. More see documentation user panel (below)

(5) Display latest images  
Display an area with latest user images. See documentation user panel (below)

(6) Number of latest images  
Display an area with latest user categories
Number of items shown in table of latest images. More see documentation user panel (below)

(7) Display user images as manageable  
Display user images as manageable like in images view.
Number of items shown in table List length defined by joomla standard length
It enables drag and drop of images like in the user images view
We do not recommend to use it. It has some disadvantages and is kept only to enable to keep the first test at some customers. It may be removed in the future. Use 'Display latest images instead'
